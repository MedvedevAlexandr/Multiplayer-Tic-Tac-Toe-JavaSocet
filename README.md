# Multiplayer-Tic-Tac-Toe-JavaSocet
Курсовая работа 2 семестра 2 курса. Реализована сетевая игра в крестики-нолики на Java сокетах. 
Игра состоит из сервера и клиента, сервер в несколько потоков обрабатывает информацию от клиентов и отправляет клиентам пакеты с информацией о ходе игры, а также обрабатывает игровую логику (действия игроков, создание игровых сессий и т.п.). В клиентсокй части реализован UI на Java Swing, пользователь может выбрать себе никнейм и создать лобби, к которому может подключиться другой игрок, или сам присоединиться к чужому лобби.
